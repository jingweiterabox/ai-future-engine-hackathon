# AI未来引擎创新马拉松 - 活动官网

这是一个为“AI未来引擎创新马拉松”活动设计的官方网站。网站采用深色科幻风格，包含动态神经网络背景、流畅的滚动动画和精致的微交互。

## 🎯 活动核心信息

- **活动名称**：AI未来引擎创新马拉松
- **核心主题**：参赛选题需围绕AI技术探索、利用AI实现业务流程自动化或提供全套AI SaaS服务展开。
- **参赛形式**：个人、团队或公司均可报名。
- **评审与奖金**：采用“参赛团队投票（占30%）+ 专家评委评审（占70%）”的混合模式，决出奖项。总奖池高达¥1,100,000人民币。

## 📅 活动时间线

- **2026年3月1日**：报名启动。官网与报名通道正式开放。
- **2026年4月15日**：报名截止。停止接收参赛申请。
- **2026年5月10日**：初赛（线上）。提交项目方案，由评委进行初审。
- **2026年6月20日**：决赛名单公布。公布进入决赛的团队。
- **2026年7月15日**：决赛路演暨颁奖典礼。现场展示与答辩，公布最终获奖名单。

## 📝 报名注册说明

- **报名费用**：每支参赛队伍（无论个人、团队或公司形式）需缴纳人民币5,000元的参赛费。
- **报名方式**：官网在线填写报名表，提交团队信息、项目构想，并完成费用支付。

## 🚀 关于我们

我们相信，AI是驱动未来商业与社会进步的核心引擎。本次创新马拉松旨在汇聚最具创造力的头脑，将前沿的AI技术转化为切实可行的商业解决方案或高效的服务工具，共同塑造智能化的未来。

## 💡 技术栈

- HTML5
- CSS3 (Tailwind CSS via CDN)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Lenis (Smooth Scroll)
- Canvas API (用于神经网络背景动画)

## 🎨 设计特点

- 深色科幻风格
- 动态神经网络背景（Canvas绘制）
- 流畅的滚动动画（GSAP + Lenis）
- 磁性按钮、光标跟随等微交互
- 响应式设计

## 📦 部署

本项目为静态网站，可直接部署到GitHub Pages、Netlify、Vercel等平台。

### GitHub Pages 部署步骤

1. 将本仓库克隆到本地
2. 确保`index.html`在仓库根目录
3. 在GitHub仓库设置中，启用GitHub Pages，选择主分支作为源
4. 访问 `https://<username>.github.io/ai-future-engine-hackathon/` 即可查看
